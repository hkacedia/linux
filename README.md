# backup

대상 디렉터리에서 일정 기간이 지난 파일을 찾아 압축하여 백업 디렉터리로 이동하고, 원본은 삭제하는 Bash 셸 스크립트입니다.

## 사용법

```bash
./backup [대상 디렉터리] [날짜]
```

- **대상 디렉터리** (생략 가능): 백업 대상 디렉터리. 생략 시 현재 디렉터리 (`.`)
- **날짜** (생략 가능): 이 일수보다 오래된 파일을 백업 대상으로 함. 생략 시 `30`일

## 사용 예시

```bash
# 현재 디렉터리에서 30일이 지난 파일 백업
./backup

# ~/Documents 디렉터리에서 30일이 지난 파일 백업
./backup ~/Documents

# ~/Downloads 디렉터리에서 60일이 지난 파일 백업
./backup ~/Downloads 60
```

## 동작 방식

1. 백업 디렉터리(`$HOME/backup`)가 없으면 자동 생성
2. 대상 디렉터리에서 지정한 날짜보다 오래된 파일을 `find`로 검색
3. 해당 파일들을 `backup_YYYYMMDD_HHMMSS.tar.gz` 형식으로 압축하여 백업 디렉터리에 저장
4. 압축이 성공하면 원본 파일을 삭제

## 백업 디렉터리 위치

스크립트 내부에 지정되어 있습니다: `$HOME/backup`

변경하려면 스크립트 상단의 `BACKUP_DIR` 변수를 수정하세요.

## 실행 권한 부여

```bash
chmod +x backup
```
